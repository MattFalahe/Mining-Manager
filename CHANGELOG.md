# Changelog

All notable changes to Mining Manager will be documented in this file.

## [1.0.0] - Unreleased

### Initial Release

- Mining ledger processing with automated price lookups
- Moon extraction tracking with jackpot detection and ready-to-fracture alerts
- Tax system using daily summaries as single source of truth
- Per-ore category tax rates (moon R4-R64, regular ore, ice, gas, abyssal)
- Guest mining detection with separate tax rates
- Event tax modifiers for mining events
- Tax code generation and wallet payment verification
- Mining events with participant tracking and leaderboards
- Reports with PDF, CSV, and JSON export
- Scheduled report generation with Discord/Slack webhook notifications
- Corporation dashboard with 12-month charts and statistics
- Theft detection and monitoring
- Configurable price cache with multiple market data sources
- 26 artisan commands with 20 automated scheduled tasks
- Full Help & Documentation page built into the plugin
- SeAT 5.x permission integration
